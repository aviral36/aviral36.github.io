FlippyTiles


I made this fun legacy tile flipping game using Python's pygame module. This is where the debate started. Unfortunately, it's an EXE file, which means for the people on non-Windows platform, too bad for you!

All you can do is hope that I get inspired to develop a LINUX based application, and for those on android, cheers! I'm learning Flutter. Who knows within a few days I'll throw in a FlippyTiles app for android. Fingers Crossed.


This is a ZIP file.
Extract the files into a new folder. The game does not require installation.
To play, run the FlippyTiles.exe
Do not delete any folder files.

Piplada


Piplada is a software that generates aesthetically pleasing symmetrical images based on the initial motion information provided by the user.
The code is entirely build on python and is available for download as an EXE file for Windows users.

So far, the entire code has been written by me. Piplada is a project under development, and the entire code is available on GitHub for those who want to dive deep into the depths of fascinating progamming.
